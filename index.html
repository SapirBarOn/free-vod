WELCOME TO FREE VOD WEB SERVICE

show all free contents(GET):

if you want to get all the free contents click on the link bellow:

https://free-vod.herokuapp.com/getAllContents

show contents by specific category(POST):

if you want to get a specific category you need use the link bellow and you can change the category name in the query string in this example the wanted category is kids:

https://free-vod.herokuapp.com/getByCategory/category=kids

show contents in specific category and minimum of views:(GET)

if you want to get contents in a specific category and minimum of views you need use the link bellow and you can change the category name and minimum number views in the query string in this example the wanted category is kids and minimum views 150:

https://free-vod.herokuapp.com/getByCategoryAndViews/category=kids&views=150
